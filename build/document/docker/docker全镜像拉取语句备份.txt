# 根据启动推荐顺序-jdk版本如按照以下搭建可以都用java21

1、RustFS 版本 【可最新】 版本只需要配合java可以链接使用即可
docker pull rustfs/rustfs:1.0.0-alpha.73

2、redis 版本 【8.4.0已测试，9未测试】 版本只需要配合java可以链接使用即可
docker pull redis:8.4.0

3、mysql 版本 【根据spring cloud alibaba-2025.0.0.0 现有支持 9.2.0】 版本需要配合spring cloud alibaba
docker pull mysql:9.2.0

4、nacos 版本 【根据spring cloud alibaba-2025.0.0.0 现有支持 v3.0.3】 版本需要配合spring cloud alibaba
docker pull nacos/nacos-server:v3.0.3

5、sentinel-dashboard 版本 【根据spring cloud alibaba-2025.0.0.0 现有支持 1.8.9】版本需要配合spring cloud alibaba
因为docker上的不支持mac所以是自己编译1.8.9的jar在sentinel-dashboard执行Dockerfile

6、rocketmq 版本 【根据spring cloud alibaba-2025.0.0.0 现有支持 5.3.1】 版本需要配合spring cloud alibaba
docker pull apache/rocketmq:5.3.1

7、mongodb 版本 【根据spring cloud alibaba-2025.0.0.0 现有支持 8.2.2】 版本需要配合spring cloud alibaba
docker pull mongo:8.2.2

8、seata 版本 【根据spring cloud alibaba-2025.0.0.0 现有支持 2.5.0.jdk21】 版本需要配合spring cloud alibaba
docker pull apache/seata-server:2.5.0.jdk21

9、elk 版本【根据spring cloud alibaba-2025.0.0.0 现有支持 8.18.8】 版本需要配合spring cloud alibaba
docker pull elasticsearch:8.18.8
docker pull kibana:8.18.8
docker pull logstash:8.18.8
docker pull elastic/filebeat:8.18.8

10、skywalking 【可最新】 版本只需要配合java可以链接使用即可
docker pull apache/skywalking-oap-server:10.3.0-java21
docker pull apache/skywalking-ui:10.3.0-java21

另本地需下载最新的 skywalking-agent 当前最新 9.5.0

11、xxl-job 【可最新】 版本只需要配合java可以链接使用即可
docker pull xxl-job-admin:3.3.2
